## 简介



该工具是使用javaFX开发的基于信息收集进行组合生成密码字典的工具，可以快速组成密码字典。



## 使用说明


直接下载releases版本即可

**使用JDK8启动，命令如下：**

```
java -jar dicttools-1.0-SNAPSHOT.jar 
```
